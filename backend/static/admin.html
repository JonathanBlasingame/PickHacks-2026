Admin: Define Parking Regions (spots + zones)

Need to place lots/cameras on the map? Use Map Setup.
Click to add points, or start with a template shape.
For zones, use exactly 4 corners. Drag vertices (or drag inside shape) to position.
Press Enter to save. Press Esc to clear current points.
If blank, spots are auto-named incrementally (e.g., Spot 1, Spot 2; Reserved 1...).
Columns = spots across for horizontal zones. For vertical zones, treat this like rows (rotation-aware split).
The tool auto-derives the other dimension from capacity.
If the virtual spots look rotated wrong, try 0, 90, -15, 15, etc.
